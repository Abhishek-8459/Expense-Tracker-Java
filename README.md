# Expense Tracker (Java)  
A simple Java-based Expense Tracker with features like adding, viewing, undoing, and sorting expenses.  

## Features  
✔ Add & View Expenses  
✔ Undo Last Expense (Stack-based)  
✔ Sort by Date or Category  

## Technologies Used  
- Java  
- Data Structures (Stacks, Lists)  
- Sorting Algorithms
